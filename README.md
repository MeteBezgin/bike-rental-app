Stack:
- MongoDB database
- ExpressJS Backend
- NextJS Frontend

## Requirements
- node

## Fetch dependencies

```
cd frontend
npm i
cd ../backend
npm i
```

## Launch frontend

```

npm start
```

The frontend will be available under `localhost:3001`.
